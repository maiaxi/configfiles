Installation guide.

First of all, you must install i3WM and for my config to work, you must instal the following programs:
blueman;
nm;
feh;
picom;
polybar;
alacritty;

Then install the following fonts:

nerd-fonts-jetbrains-mono;
ttf-jetbrains-mono;
ttf-iosevka-nerd;
ttf-nerd-fonts-symbols-1000-em-mono;
